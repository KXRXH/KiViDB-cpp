# KiViDB C++

Simple non-SQL database. Was written with [Golang](https://github.com/BigCubeCat/KiViDB), but now with C++

## Usage

### Create an environment variables

```bash
$ export KIVIDB_HOME=/any/folder/you/like
$ export KIVIDB_PORT=8080
```

### Build

```bash
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
```

### Run

```bash
./kividb
```
